Aversive++

Aversive++ is an Open Source C++ library aimed to ease the development of applications on AVR-based and ARM-based microcontrollers.

Why using Aversive++

This project provides an interface simple enough to create complex applications, and optimized enough to enable small microcontrollers to execute these applications (less than 4K!).

Control systems

Control easily motors, and robotic systems with pre-implemented controllers (PID, bang-bang, ...).

Communication

The library provides a standard C++ API to communicate with others systems.

Memory security

With Aversive++, you can use most common containers, specially designed for embedded systems.

The C++ philosophy

"Unused features should not negatively impact created executables"

Multiplatform

Thanks to Aversive++ library, the same application can be compiled for AVRs, ARMs, and the simulator named SASIAE.

Compile-time verification

The library warns you if you use badly your hardware equipment.
