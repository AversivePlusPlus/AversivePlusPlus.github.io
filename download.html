Aversive++

This is the Aversive++ library's tags list. Each tag name refers to the month it has been released.
v14.11 - download sources

The v14.06 is the version used by the team Eirbot at the French division of Eurobot in 2014. Only the most needed features are available in this version, and only one microcontroller is completely supported.

v14.06 - download sources

The v14.11 is the version used by the team Astrabot at the French division of Eurobot in 2015. A lot of architecture-independent features are available. A begining of STM32 support is available, but it is not reliable enough.

Visit the Github Repository to submit feature request and bugs.

tinySASIAE

tinySASIAE is a light-weight simulator that features a simple two-wheeled robot simulation. It can be used to teach how to control the movement of a mobile robot. Each version of the simulator is intended to work with Aversive++ in the same version.
